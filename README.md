# Scoop Bucket
A personal [Scoop](https://scoop.sh) bucket for my own apps.

## Usage

```pwsh
scoop bucket add nikirack https://github.com/nikirack/bucket
scoop install nikirack/<appname>
```

## Apps
[todo.tui](https://github.com/nikirack/todo.tui): A terminal-based [todo.txt](https://todotxt.org/) manager with a TUI
